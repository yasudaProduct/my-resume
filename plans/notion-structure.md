# Notion データベース構成

Notion 側に以下の 2 つのデータベースを作成してください。

---

## 1. 履歴書 (Resume) データベース

**データベース名:** `履歴書`

### プロパティ一覧

| プロパティ名 | 型 | 説明 | 例 |
|---|---|---|---|
| 氏名 | Title | 氏名 (データベースのタイトル列) | 山田 太郎 |
| フリガナ | Text | 氏名のフリガナ | ヤマダ タロウ |
| 生年月日 | Date | 生年月日 | 1990-01-15 |
| 住所 | Text | 現住所 | 東京都渋谷区... |
| 電話番号 | Phone | 電話番号 | 090-1234-5678 |
| メールアドレス | Email | メールアドレス | taro@example.com |
| 最寄り駅 | Text | 最寄り駅 | JR山手線 渋谷駅 |
| 顔写真 | Files | 証明写真 (任意) | — |

### 子ページ (ページ本文に記載)

データベースのエントリ (1行) をページとして開き、本文に以下を **見出し (H2)** で区切って記載してください。

```
## 学歴・職歴
- 2009年4月 ○○大学 入学
- 2013年3月 ○○大学 卒業
- 2013年4月 株式会社○○ 入社
- 2018年3月 株式会社○○ 退社

## 資格・免許
- 2012年6月 基本情報技術者試験 合格
- 2015年11月 TOEIC 850点

## 自己PR
ここに自己PRを記載...

## 趣味・特技
ここに趣味・特技を記載...
```

## 2. 職務経歴書 (Career) データベース

**データベース名:** `職務経歴書`

### プロパティ一覧

| プロパティ名 | 型 | 説明 | 例 |
|---|---|---|---|
| 会社名 | Title | 会社名 (データベースのタイトル列) | 株式会社○○ |
| 在籍期間 | Text | 開始〜終了 | 2013年4月〜2018年3月 |
| 雇用形態 | Select | 雇用形態 | 正社員 / 契約社員 / 業務委託 |
| 役職 | Text | 最終役職 | リードエンジニア |
| 事業内容 | Text | 会社の事業内容 | Webサービスの企画・開発 |
| 従業員数 | Text | 会社規模 | 約200名 |
| 並び順 | Number | 表示順 (新しい順に小さい数字) | 1 |

### 子ページ (ページ本文に記載)

各エントリのページ本文に、プロジェクトや業務内容を記載してください。

```
## 職務要約
Webアプリケーション開発のリードエンジニアとして...

## プロジェクト: ○○システムのリニューアル
- **期間:** 2016年4月〜2017年9月
- **チーム規模:** 5名
- **役割:** テックリード
- **担当業務:**
  - 技術選定・アーキテクチャ設計
  - フロントエンド・バックエンドの実装
  - コードレビュー・メンバー育成
- **使用技術:** React, Node.js, PostgreSQL, AWS
- **成果:**
  - ページ読み込み速度を60%改善
  - リリース後の障害件数を前年比50%削減

## プロジェクト: ○○機能の新規開発
...

## 使用技術・スキル
- **言語:** TypeScript, Python, Go
- **フレームワーク:** React, Next.js, Django
- **インフラ:** AWS (EC2, ECS, RDS, S3), Docker
- **その他:** Git, CI/CD, Agile/Scrum
```

## Notion Integration の設定手順

1. https://www.notion.so/my-integrations にアクセス
2. 「新しいインテグレーション」を作成
3. 名前を付けて (例: `my-resume`)、ワークスペースを選択
4. 「Internal Integration Secret」をコピー → GitHub Secrets に `NOTION_API_KEY` として登録
5. 上記 2 つのデータベースそれぞれで「コネクト」からインテグレーションを追加
6. 各データベースの URL からデータベース ID を取得 → GitHub Secrets に登録
   - `NOTION_RESUME_DB_ID` (履歴書)
   - `NOTION_CAREER_DB_ID` (職務経歴書)

### データベース ID の確認方法

Notion データベースの URL:
```
https://www.notion.so/xxxxxxxx?v=yyyyyyyy
                      ^^^^^^^^
                      この部分がデータベース ID
```
