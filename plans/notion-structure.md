# Notion データベース構成

Notion 側に以下の **6 つ** のデータベースを作成してください。

---

## 1. 履歴書 (Resume) データベース

**データベース名:** `履歴書`

| プロパティ名 | 型 | 説明 | 例 |
|---|---|---|---|
| 氏名 | Title | 氏名 (タイトル列) | 山田 太郎 |
| フリガナ | Text | 氏名のフリガナ | ヤマダ タロウ |
| 生年月日 | Date | 生年月日 | 1990-01-15 |
| 住所 | Text | 現住所 | 東京都渋谷区... |
| 電話番号 | Phone | 電話番号 | 090-1234-5678 |
| メールアドレス | Email | メールアドレス | taro@example.com |
| 最寄り駅 | Text | 最寄り駅 | JR山手線 渋谷駅 |
| 顔写真 | Files | 証明写真 (任意) | — |
| 自己PR | Text | 自己PR | 私は... |
| 趣味・特技 | Text | 趣味・特技 | 読書、ランニング |

> ※ 1 行だけ登録してください (自分の情報)。

## 2. 学歴職歴 データベース

**データベース名:** `学歴職歴`

| プロパティ名 | 型 | 説明 | 例 |
|---|---|---|---|
| 内容 | Title | 学歴・職歴の内容 (タイトル列) | ○○大学 入学 |
| 年月 | Date | 年月 | 2009-04-01 |
| 区分 | Select | 学歴 or 職歴 | 学歴 / 職歴 |
| 並び順 | Number | 表示順 (小さい順) | 1 |

**登録例:**

| 内容 | 年月 | 区分 | 並び順 |
|---|---|---|---|
| ○○大学 入学 | 2009-04-01 | 学歴 | 1 |
| ○○大学 卒業 | 2013-03-01 | 学歴 | 2 |
| 株式会社○○ 入社 | 2013-04-01 | 職歴 | 3 |
| 株式会社○○ 退社 | 2018-03-01 | 職歴 | 4 |

## 4. 職務経歴書 (Career) データベース

**データベース名:** `職務経歴書`

| プロパティ名 | 型 | 説明 | 例 |
|---|---|---|---|
| 会社名 | Title | 会社名 (タイトル列) | 株式会社○○ |
| 在籍期間 | Date | 開始〜終了 (日付範囲で設定) | 2013-04-01 → 2018-03-01 |
| 雇用形態 | Select | 雇用形態 | 正社員 / 契約社員 / 業務委託 |
| 役職 | Text | 最終役職 | リードエンジニア |
| 事業内容 | Text | 会社の事業内容 | Webサービスの企画・開発 |
| 従業員数 | Text | 会社規模 | 約200名 |
| 職務要約 | Text | 会社での職務概要 | Webアプリ開発のリードエンジニアとして... |
| 並び順 | Number | 表示順 (新しい順に小さい数字) | 1 |

## 5. プロジェクト データベース

## Notion Integration の設定手順

1. https://www.notion.so/my-integrations にアクセス
2. 「新しいインテグレーション」を作成
3. 名前を付けて (例: `my-resume`)、ワークスペースを選択
4. 「Internal Integration Secret」をコピー → GitHub Secrets に `NOTION_API_KEY` として登録
5. 上記 6 つのデータベースそれぞれで「コネクト」からインテグレーションを追加
6. 各データベースの URL からデータベース ID を取得 → GitHub Secrets に登録
   - `NOTION_RESUME_DB_ID` (履歴書)
   - `NOTION_EDUCATION_WORK_DB_ID` (学歴職歴)
   - `NOTION_CERTIFICATION_DB_ID` (資格)
   - `NOTION_CAREER_DB_ID` (職務経歴書)
   - `NOTION_PROJECT_DB_ID` (プロジェクト)
   - `NOTION_SKILL_DB_ID` (スキル)

### データベース ID の確認方法

Notion データベースの URL:
```
https://www.notion.so/xxxxxxxx?v=yyyyyyyy
                      ^^^^^^^^
                      この部分がデータベース ID
```
